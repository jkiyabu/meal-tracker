# Meal Tracker

This application will allow the user to track their meals throughout the day.  They can add and edit meals, and view them based on the amount of calories they have.

## Installation

* `git clone <repository-url>`
* `cd meal-tracker`
* `npm install`
* `bower install`

## Running / Development

* `gulp build`
* `gulp serve`
